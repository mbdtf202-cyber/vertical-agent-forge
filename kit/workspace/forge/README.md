# Forge Workspace

This directory is the durable evidence trail for self-improvement.

Use it like this:

- `cases/inbox/`
  - raw incoming problems and opportunities
- `cases/active/`
  - problems currently being improved
- `cases/accepted/`
  - closed cases with successful rollout
- `cases/rejected/`
  - closed cases that should not ship
- `candidates/`
  - worker manifests and proposed changes
- `decisions/`
  - promoter decisions
- `eval/`
  - regression, adversarial, and evaluation reports
- `memory/`
  - durable lessons that should influence future work
- `releases/`
  - rollout notes and rollback plans
- `monitoring/`
  - post-promotion watch notes
- `templates/`
  - starter artifacts

Every active case should make it obvious whether the forge is:

- advancing
- blocked
- holding
- waiting on the next wake
