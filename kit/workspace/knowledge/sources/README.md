# Knowledge Sources

Place raw business materials here.

Recommended subdirectories:

- `policies/`
- `glossary/`
- `actions/`
- `evals/`
- `routing/`
- `tickets/`
- `catalog/`
- `macros/`

The compiler reads this directory and writes normalized outputs into:

- `knowledge/domain/compiled/`
- `knowledge/policies/`
- `knowledge/glossary/`
- `knowledge/action-catalog/`
- `knowledge/evals/`
- `knowledge/routing/`
