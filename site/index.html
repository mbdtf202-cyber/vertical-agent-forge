Build, validate, ship, and operate a vertical expert agent.

Factory roles

Separate delivery, compilation, evaluation, deployment, and monitoring instead of hiding everything in one prompt.

  • app-main serves the user
  • app-domain-compiler compiles the domain pack
  • app-evaluator, app-deployer, and app-observer close the loop

Compiled domain system

Raw business materials become compiled policies, glossary, action catalog, eval seeds, routing rules, and metric definitions.

  • Business truth lives in files, not only transcripts
  • Compiler outputs are deterministic and testable
  • Reference SaaS Support pack ships end to end

Control-plane automation

Every change passes through validation, rollout evidence, connector checks, promotion gates, and breach-triggered rollback.

  • Snapshot, jobs, metrics, connectors, incidents
  • Strict activation by default
  • Canary-first deployment with auto rollback

Comparison

What makes this a product, not a prompt hack

Single Prompt Agent

  • One runtime does everything
  • Quality rules drift over time
  • Hard to audit regressions

Ad-hoc Multi-Agent

  • Better separation, weak release discipline
  • Artifacts are inconsistent
  • Promotion logic is often implicit

Vertical Agent Forge

  • Compiled domain pack plus runtime control plane
  • Bounded roles with explicit gates and anti-stall pivots
  • Reusable artifacts, memory, release assets, and rollback logic

Examples

One-click domain templates

SaaS Support

Billing policy, refunds, plan eligibility, ticket fixtures, and metric thresholds.

node ./bin/vertical-agent-forge.mjs bootstrap --domain saas-support

Travel Concierge

Shortlist-first destination guidance and route tradeoffs.

node ./bin/vertical-agent-forge.mjs init --domain travel-concierge

Research Ops

Evidence synthesis, triage, and internal analyst workflows.

node ./bin/vertical-agent-forge.mjs init --domain research-ops

CLI Lifecycle

Bootstrap, validate, evaluate, ship, rollback

node ./bin/vertical-agent-forge.mjs bootstrap --domain saas-support
node ./bin/vertical-agent-forge.mjs doctor --deep
node ./bin/vertical-agent-forge.mjs validate
node ./bin/vertical-agent-forge.mjs run-evals --case CASE-20260317-001 --stage canary
node ./bin/vertical-agent-forge.mjs deploy --case CASE-20260317-001 --stage live
node ./bin/vertical-agent-forge.mjs rollback --case CASE-20260317-001 --reason "canary breach"
node ./bin/vertical-agent-forge.mjs status --deep
