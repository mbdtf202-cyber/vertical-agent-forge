Production Toolkit For OpenClaw

Run a vertical agent like a product, not a prompt.

Vertical Agent Forge adds a durable improvement loop around your OpenClaw deployment with bounded role agents, release gates, artifact contracts, and hot-plug installation.

Purpose-built roles

Split user delivery from quality control. Keep one runtime focused on users and another on improvement.

  • app-main handles the customer conversation
  • app-forge owns the loop
  • Subagents stay bounded and accountable

Durable orchestration

Use the OpenClaw runtime you already have: task, continuous-worker, wakes, and subagents.

  • Progress persists across runs
  • Wake schedules survive beyond one chat turn
  • Artifacts stay in files, not only in transcripts

Release discipline

Every change passes through proposal, critique, adversarial review, and promotion.

  • Candidate artifacts
  • Scorecards and attack reports
  • Decision and distillation records
