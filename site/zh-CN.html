OpenClaw 生产级工具包

把垂直 agent 做成产品，而不是大 prompt。

Vertical Agent Forge 为 OpenClaw 增加一个 durable improvement loop： 多角色 agent、release gate、artifact 合同，以及热插拔安装方式。

下载发布包 查看仓库
  • 直接热插拔到现有 OpenClaw 环境
  • Worker / Critic / Adversary / Promoter / Archivist
  • durable task 与 wake 调度
app-main 用户交付
app-forge 持续改进
release gate 先证据，后晋升

角色清晰

把用户交付和质量控制拆开，不让一个 agent 同时承担两种责任。

  • app-main 面向用户
  • app-forge 编排改进
  • 子 agent 保持有边界、可追责

持久化编排

直接复用 OpenClaw 现有 runtime：task、 continuous-worker、wake、subagent。

  • 进度跨多轮持久存在
  • wake 不依赖单次聊天
  • 核心证据留在文件里

发布纪律

每个改动都要经过 proposal、critique、adversarial review 和 promotion。

  • candidate artifact
  • scorecard 与 attack report
  • decision 与 distillation record
